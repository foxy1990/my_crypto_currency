# my_cc

Convert cryptocurrency to Japanese Yen and Dollar.<br>
Bitcoin, Ethereum, XRP, NEM, BitZeny, MonaCoin, ALIS, Sakuracoin, KumaCoin, Fujicoin, Koto.<br>
Copy the after convert currency to the clipboard.<br>
